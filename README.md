# CricStars
* Developed a responsive multi-page e-commerce website for cricket equipment using HTML, CSS and JavaScript, featuring Home, Products, Cart, Authentication, Contact and About web pages. 
* Implemented interactive functionalities including product search, filtering, sorting, shopping cart management and client-side form validation using JavaScript and Local Storage. 
* Designed a modern UI with CSS Grid, Flexbox, responsive layouts, animations and reusable components, ensuring compatibility across desktop, tablet and mobile devices.
